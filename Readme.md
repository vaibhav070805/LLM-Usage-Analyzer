# LLM Usage Analyzer

## Note :-This the copy of the repositiory in order to hide API key privacy.

## Overview
A Chrome extension that analyzes how users interact with LLMs and provides real-time usage feedback.

## Problem
Lack of feedback mechanisms encouraging responsible learning in LLM usage.

## Solution
Observes prompt characteristics and user intent to increase awareness during LLM interactions.

## Features
- Prompt capture across LLM platforms
- Prompt length and complexity analysis
- Direct code request detection
- Real-time popup usage metrics
- Exportable prompt metrics

## Architecture
Browser extension with prompt capture, behavior analysis, and local logging modules.

## Privacy
All analysis runs locally. No background data uploads.
